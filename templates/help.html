üöÄ Getting Started
Welcome to Job Application Automation!

This tool helps you automatically search for jobs and apply to them across multiple job portals.

First Steps:
  1. Complete your profile with accurate information
  2. Upload your resume in PDF format
  3. Set your job search preferences
  4. Test the system with a manual search
  5. Set up automated scheduling
Important: Always review and customize your profile before starting automation.
üë§ Profile Setup
Required Information:
  • Name: Your full name as it appears on your resume
  • Email: Professional email address
  • Phone: Valid Indian mobile number
  • Location: Your preferred work location
  • Skills: Key technical skills relevant to your target roles
  • Experience: Years of relevant experience
  • Expected Salary: Your salary expectations in INR
Resume Tips:
  • Upload PDF format only
  • Keep file size under 2MB
  • Use ATS-friendly format
  • Include relevant keywords for AI/ML roles
  • Update regularly with new skills and projects
üîç Job Search
Search Filters:
  • Target Roles: Select specific job titles you're interested in
  • Experience Range: Set minimum and maximum years of experience
  • Salary Range: Set minimum expected salary
  • Location: Choose remote, specific city, or multiple locations
  • Job Portals: Select which platforms to search
Supported Job Portals:
  • LinkedIn: Professional networking
  • Naukri.com: India's largest job portal
  • Indeed: Global job search
  • Wellfound: Startup jobs
  • RemoteOK: Remote opportunities
  • Company Websites: Direct applications
Note: Some portals may require manual login for full functionality.
ü§ñ Automation
Scheduling Options:
  • Daily: Search and apply every day at specified time
  • Weekly: Run once per week
  • Manual: Run on-demand only
Safety Features:
  • Application limits per day (default: 5)
  • Rate limiting between requests
  • Duplicate application prevention
  • Error handling and recovery
Best Practices:
  • Start with manual testing
  • Monitor application success rates
  • Adjust search criteria based on results
  • Keep your profile updated
  • Review applications regularly
